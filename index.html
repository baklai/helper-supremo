Supremo - Инструкция по настройке удаленного доступа

Supremo - это инструмент для удаленного управления другим компьютером, который станет хорошим помощником для обеспечения технической поддержки по интернету, отображения своего экрана другим пользователям и т.д.

Запуск программы без установки

Приложение не требует установки в систему и сразу готово для использования. При этом не требуется настраивать фаервол или роутер.
